# Perceptron

Понятием перцептрон обозначают разные понятия:

1. Алгоритм обучения с учителем для двоичных классификаторов
1. модель отдельного нейрона
1. модель однослойной нейронной сети
1. модель много слойной нейронной сети

## Отдельный нейрон

Это искусственный нейрон, функция активации которого устроена сложнее, чем у бинарного порогового нейрона. Сигналы на входе теперь не 0 или 1, а вещественные числа. Функция активации - это не просто сумма, но значению на каждом входе добавляется множитель - **вес**. В итоге для определения значения функция активации использует **взвешенную сумму** входных сигналов.
Иногда к взвешенной сумме добавляют константу, которую называют **bias**. Это решает некоторые проблемы при обучении.

Вычисление взвешенной суммы происходит, когда выполняется операция умножения матрицы на вектор.

![Weights and vector](img/Weights%20and%20vector.png) [2]

> ..перцептрон может обучаться. Этот процесс заключается в автоматическом подборе значений весов для каждого входного сигнала. Алгоритм обучения основан на теории Хебба.

В 1958г Фрэнк Розенблат сконструировал первый нейрокомпьютер Mark I Perceptron. Он представлял собой однослойную нейронную сеть, предназначался для распознавания изображений и состоял из аналоговых компонентов: нейроны представляли собой электромеханические устройства. На входную матрицу из фотодетекторов подавалось изображение. Результат работы анализировлся человеком и после этого производилась корректировка весов.

## Однослойная нейронная сеть

Впоследствии класс подобных моделей стал называться *feedforward neural network* (FNN).

> Перцептрон создает набор ассоциаций (А-элементы) между входными сигналами (S-элементы) и реакцией на них (R-элементы)

А-элементы формируют скрытый слой, в однослойной сети он один.
FNN используются в [[Transformers]] как часть *Transformer block*.

## Convolutional neural network

Свертночная нейронная сеть - это подвид FNN и оптимизирована для обработки изображений.
